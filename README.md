# Juicy_Delights
Juicy Delights Business: Comprehensive Business Analysis Project

A client is opening a Juice and Smoothie takeout shop and have been tasked with creating a relational database and a performance monitoring dashboard.

<b> 𝐎𝐫𝐝𝐞𝐫𝐬: Track item details, prices, quantities, and customer delivery information.
<b>𝐒𝐭𝐨𝐜𝐤 𝐂𝐨𝐧𝐭𝐫𝐨𝐥: Monitor ingredients, inventory levels, and ingredient costs.
<b> 𝐒𝐭𝐚𝐟𝐟: Manage staff details, hourly rates, work hours, and wages.

 ## 𝐏𝐫𝐨𝐣𝐞𝐜𝐭 𝐇𝐢𝐠𝐡𝐥𝐢𝐠𝐡𝐭𝐬:
